# CalculatorGUI

##Description
-A Calculator program built using Java(Swing and AWT)

##Features
- Performs basic arithematic operation (+,-,/,*,%)
- Deals Multi Operands at the same time
- Uses Operator precedence
- Provides Graphical user Interface
